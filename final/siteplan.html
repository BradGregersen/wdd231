Site Overview

Site Name: Computing History Archive

I chose this name because it clearly identifies the subject matter (computing) and the nature of the site (a historical archive). It is appropriate for students, educators, and technology enthusiasts.

Purpose: This site provides a comprehensive look at computing history by offering an interactive timeline of milestones and profiles of key pioneers for students, educators, and technology enthusiasts.

Scenarios

  • Scenario 1: "Who invented the first programmable computer?"
  • Scenario 2: "What was the first instance of a computer, mechanical or electronic, in the world?"

Data Structure

The site will use two JSON data sources:

Pioneers Data

  • name — Full name of the pioneer
  • birthYear — Year of birth
  • deathYear — Year of death (null if living)
  • image — Path to image file
  • contribution — What they created or invented
  • quote — A famous quote from them

Milestones Data

  • year — Year the event occurred
  • title — Name of the event/milestone
  • description — What happened
  • impact — Why it matters
  • category — Type (hardware, software, networking, consumer)
